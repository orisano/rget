# rget
rget is the parallel downloader for GitHub release.

## Installation
```bash
go get github.com/orisano/rget
```

## How to use
```
$ rget
Usage of rget:
  -o string
    	output file path (required)
  -u string
    	url (required)
  -b int
    	block size (MB)
  -P int
    	maxprocs (default 4)
  -x	add executable flag
  -v	show verbose
  -V	show version
```
